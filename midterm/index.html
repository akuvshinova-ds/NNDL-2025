3. Preprocessing

Preprocessing Steps:
  • Drop Useless Features: Remove Dt_Customer, Days_Customer, source columns
  • Remove Extreme Outliers: Filter Year_Birth > 1920 and Income < 100,000
  • Imputation: SimpleImputer for missing values (median for numeric, most_frequent for categorical)
  • Scaling: StandardScaler for numerical features
  • Encoding: OneHotEncoder for categorical features (Education, Marital_Status)
  • Oversampling: RandomOverSampler to balance training data

4. Model Setup

5. Training

Training Configuration:
